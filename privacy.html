We comply with applicable data protection laws including GDPR (for EU/EEA users), CCPA (for California residents), and other applicable privacy regulations.

EU/EEA Users (GDPR)

  • Right to restrict processing

To exercise any right, email privacy@prestocks.io or use the self-service tools in Settings → Privacy.

8. Security Measures

  • Passwords hashed with bcrypt (never stored in plaintext)
  • Data encrypted in transit (TLS 1.3) and at rest (AES-256)
  • Two-factor authentication available
  • Regular security audits and penetration testing
  • Access controls with principle of least privilege
  • Automated anomaly detection for unauthorized access attempts

Despite these measures, no system is 100% secure. Report vulnerabilities to security@prestocks.io.

10. Data Retention Policy

We retain personal data only as long as necessary for the purposes outlined in this policy. Our retention principles:

  1. Minimization: We collect only what is needed for service delivery
  2. Purpose limitation: Data is used only for stated purposes
  3. Time limitation: Data is deleted or anonymized when no longer needed
  4. User control: Users can request deletion at any time

Deletion Process

  1. User initiates deletion (Settings → Account → Delete Account)
  2. 30-day grace period (account deactivated, recoverable)
  3. After 30 days: personal data hard-deleted from primary systems
  4. After 60 days: removed from all backups
  5. Anonymized aggregate data (e.g., total user counts) may be retained indefinitely

Exceptions

We may retain data beyond standard periods when required by:

  • Active legal proceedings or investigations
  • Regulatory requirements (e.g., financial record-keeping)
  • Fraud prevention (limited data for blocked accounts)

11. International Transfers

Data may be processed in the United States. For EU/EEA transfers, we rely on Standard Contractual Clauses (SCCs) or equivalent safeguards.

12. Children's Privacy

PreStocks is not intended for users under 18. We do not knowingly collect data from minors. If we learn a user is under 18, we will promptly delete their account.

13. Changes to This Policy

We may update this policy periodically. Material changes will be communicated via email and an in-app banner at least 30 days before taking effect.
